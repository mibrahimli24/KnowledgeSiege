Knowledge Siege 🏰

Knowledge Siege is a desktop-based strategy/quiz game built using Java Swing. The project focuses on implementing core Object-Oriented Programming (OOP) principles, event-driven programming, and a dynamic graphical user interface.

🚀 Features

Interactive GUI: Built entirely with Java Swing and AWT components.

Game Logic Engine: Manages player states, scoring systems, and win/loss conditions.

Asset Management: Custom handling of game sprites and visual elements.

Event-Driven Design: Efficiently handles user inputs and real-time game updates.

🛠️ Tech Stack

Language: Java

Framework: Java Swing / AWT

IDE: IntelliJ IDEA

Version Control: Git

📦 Installation & Running

To run this project locally, ensure you have JDK 11 or higher installed.

Clone the repository:

Bash
git clone https://github.com/mibrahimli24/KnowledgeSiege.git
Open in your IDE: * Open IntelliJ IDEA.

Select File > Open and navigate to the cloned folder.

Run the application:

Locate the Main.java (or your primary entry class) and click the green Run button.
